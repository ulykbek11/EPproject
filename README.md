EduPath is a modern web application built with React, Vite, and TypeScript. It leverages a powerful tech stack including Tailwind CSS, Shadcn UI, Supabase, and Google Generative AI to provide an intelligent, interactive, and responsive user experience.

Tech Stack

Core Technologies
Framework: React 18
Build Tool: Vite
Language: TypeScript
Styling: Tailwind CSS

State Management and Data Fetching
Server State: @tanstack/react-query
Routing: react-router-dom
Form Handling: react-hook-form with @hookform/resolvers and zod for validation

UI and Components
Component Library: Shadcn UI built on top of @radix-ui primitives
Icons: lucide-react
Animations: tailwindcss-animate and framer-motion via UI components
Charts: recharts
Markdown Rendering: react-markdown with remark-gfm

Backend and AI Integrations
Database and Auth: @supabase/supabase-js
AI Capabilities: @google/generative-ai

Utilities and Document Processing
PDF Processing: pdfjs-dist
Word Document Processing: mammoth
Date Formatting: date-fns

Prerequisites

Before you begin, ensure you have met the following requirements
Node.js version 18 or higher recommended
npm or yarn or pnpm

Installation and Setup

Clone the repository if you have not already
Install dependencies using npm install
Environment Variables Configuration
The .env file is intentionally excluded from version control for security
Use the provided .env.example file as a template to create your local .env file using cp .env.example .env
Fill in the required API keys and configuration values in the .env file such as Supabase URL and Key and Google AI Key

Available Scripts

npm run dev
Starts the Vite development server with Hot Module Replacement

npm run build
Compiles TypeScript and builds the app for production to the dist folder with optimized performance

npm run build dev
Builds the app using the development mode configuration

npm run lint
Runs ESLint to analyze the code for potential errors and stylistic issues

npm run preview
Locally previews the production build generated by npm run build
